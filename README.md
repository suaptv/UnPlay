UnPlay 是一款专业的 DLNA 接收器(不支持AirPlay)与场景体验应用，让你的 Apple TV 不再只是播放器，而是家庭中的氛围创造者。
无论是投屏、时钟、天气、壁纸，还是动态场景展示，都能在 UnPlay 中一体呈现。

此外，在繁忙的生活中，UnPlay也能给您带来宁静，让您享受安静聆听的时刻。 https://apps.apple.com/app/unplay/id6450034641

### 稳定强大的 DLNA 投屏引擎
基于 MPV 播放核心打造，适配多种复杂格式，兼容性远超普通播放器。
支持与 HomePod 协同播放，为你的家庭影院带来更自然的声画体验。
网页端也能推送视频到 Apple TV，并可组合成播放列表。

### 时钟 · 天气 · 壁纸 · 多场景展示
你的电视屏幕，不止可以播放，还能“展示”。
丰富精美的时钟主题
- 复古翻页时钟
- 动态渐变数字时钟
- 模拟指针时钟
- 壁纸时钟（天气背景 / 渐变背景）
- 极简时钟
- 方块时钟
- LED 数字怀旧时钟

### 天气场景
- 接入 Apple Weather，展示准确高质量的天气信息
- 适用于客厅、小店、前台、大堂等作为“氛围屏”

### 动态 Top 小组件 
- 动态壁纸
- 视频相框
- 壁炉动画壁纸
- 纪念日小组件（倒计时 / 纪念日显示）
- [帮助] UnPlay--设置(下滑或按下方向键)--Top小组件 可以选择海报轮播、大横幅、方卡片、竖卡片

### 系统工具与高级定制
- 查看 Apple TV 空间使用情况
- 支持多款个性化 App 图标主题
- 多种壁纸主题与场景快速切换

### DLNA接收器
- HTTP推送：网页端推送视频, 组合成列表播放
- homepod和天猫精灵支持下一集歌曲(点播、直播、歌曲), atv必备技能
- 单集循环播放(儿歌、音乐、MV首选)
- 播放器下滑(倍速、选择音频、字幕轨道、翻转画面、比例切换、循环列表)
- 视频旋转(默认，90度，180度，270度)
- 连续循环，单个循环，列表循环
- 画面比例(拉伸填充，等比缩放，自定义缩放[随意调节，针对老剧])
