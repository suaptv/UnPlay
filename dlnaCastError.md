## 1️⃣ HDR 无法激活怎么办

### ✅ 先确认视频是否带 HDR 标记
- 📌 **DLNA投屏的视频必须包含 HDR 标记**（色彩原色 BT.2020）
- 🔍 如何查看：
  - UnPlay → **设置** → 打开 **「检查器」**
  - 播放视频时查看是否有 `BT.2020` 标记

### ✅ tvOS 设置
- 📺 **必须开启动态范围匹配**
tvOS → 设置 → 视频和音频 → 匹配内容
→ 打开「匹配动态范围」


### ⚠️ 常见问题

| 现象 | 原因 | 解决方法 |
|------|------|----------|
| 客户端投屏时显示HDR但实际没有 | 客户端只传了清晰度（2K/4K/1080），没传HDR | 换客户端或源 |
| 网盘视频没HDR | 网盘在线播放默认转码 | 用第三方客户端(比如FileBall)选**原画**投屏（带BT.2020） |
| 不确定UnPlay是否支持HDR | 无法验证 | 自己建HTTP服务器，推送本地HDR视频测试 |

> 💡 **自建验证方法**：电脑开HTTP服务器 → HTTP投屏HDR视频 → 手机浏览器打开unplay的http网址推送 → 投屏到UnPlay

## 3️⃣ 网盘投屏发紫问题

### ❓ 现象
- 投屏杜比视频 → 画面**变紫/变绿**

### 🔍 原因
- 云盘**在线转码**：把杜比视频转成了普通 SDR 格式
- 投出去的根本不是杜比片源，而是转码后的 SDR 绿片

### ✅ 解决方法
- **目前无解**（除非云盘支持原画投屏）
- 替代方案：用 Infuse, FileBall, VidHub, SenPlayer等播放器投屏

## 4️⃣ 投屏出现 403 报错

### 常见原因

| 原因 | 说明 |
|------|------|
| 🚫 **VOD限制外网** | 视频源禁止外网播放 |
| 🚫 **地域限制** | 视频源禁止中国大陆IP访问 |
| 🚫 **资源失效** | 网站上的视频文件已丢失 |
| 🚫 **客户端限制** | App主动禁止DLNA投屏 |

### ✅ 解决方法
1. 换个视频源试试
2. 换个客户端试试
3. 开VPN或 关闭VPN试试（如果是地域限制）
4. 放弃（如果是App禁投屏）

> 📌 **403 ≠ 你的问题**，很多时候是视频源本身限制

## 5️⃣ 如何查看视频分辨率

### 步骤一：设置中常开检查器
UnPlay → 设置 → 打开「检查器」

### 步骤二：播放器内查看
1. 播放视频时
2. Siri遥控器 **向下滑**（或按方向键下）
3. 找到 **「检查器」**（需先在 UnPlay 设置中打开）

之后所有视频都会显示详细参数（分辨率、码率、色彩空间等）

### 查看的内容包括：
- ✅ 分辨率（1080p/4K/8K）
- ✅ 帧率
- ✅ HDR标记
- ✅ 音频格式

## 6️⃣ 投屏只有画面没有声音

### ❓ 现象
- 投屏成功
- 有画面
- 没有声音

---

### 🔍 常见可能性一：误用了 AirPlay

很多用户误点了 AirPlay 投屏，而不是 DLNA。

⚠️ UnPlay 使用的是 DLNA 投屏方式，不是 AirPlay。

### ✅ 正确方式
- 在客户端点击 DLNA 图标
- 选择 UnPlay
- 不要点击 AirPlay 设备

如果看到的是系统弹出的 AirPlay 设备列表，那就是用错方式了。

---

### 🔍 常见可能性二：HomePod 没有出声

有时音频输出被切换，导致 HomePod 没有声音。

### ✅ 解决方法
1. 重启 Apple TV
2. 进入 tvOS：
   设置 → 视频和音频 → 音频输出
3. 手动选择 HomePod 作为音频输出设备
4. 再次尝试播放

---

### 🔍 常见可能性三：DLNA 播放器解码兼容问题

部分视频的音频编码（例如某些 AAC/AC3/EAC3 等）在默认播放器下可能会出现“有画面无声音”。

### ✅ 解决方法
- 打开 UnPlay → 设置
- 找到 **DLNA 播放器**
- 将播放器切换为 **FFmpeg**
- 返回重新投屏播放

## 📌 一句话总结

| 问题 | 核心解决方法 |
|------|-------------|
| **没HDR** | 开匹配动态范围 + 检查BT.2020标记 |
| **画面发紫** | 阿里云盘转码问题，换第三方播放器投屏 |
| **403报错** | 视频源限制，换源或放弃 |
| **不知道分辨率** | 下滑打开检查器 |
| **想网页投屏** | 用 Focus/悟空/Alook/夸克 |
| 误点 AirPlay | 改用 DLNA 投屏 |
| HomePod 无声 | 重启 Apple TV + 重新选择音频输出 |
